# Weekly-tasks for Programming and Scripting module

## Weekly Task 1 - helloworld.py
A file that contains a python program that displays Hello World when it is run.

## Weekly Task 2 - bank.py
The program prompts the user and reads in two amounts (in cents).It will add the amount together,and store the total as an integer, then it prints out the answer in a readable format with the euro sign and decimal point.

## Weekly Task 3 - accounts.py
The program reads in a 10 character account number and outputs the account number with only the last 4 digits showing, the digits before are hidden by Xs

## Weekly Task 4 - collatz.py
The program asks the user to input any positive integer and outputs the successive values of the calculation.At each step it calculates the next value and if its even, divide it by two, but if its odd, multiply it by three and add one.

## Weekly Task 5 - weekday.py
The program outputs whether or not today is a weekday or week end.

## Weekly Task 6 - squareroot.py
This program takes a positive floating point number as input and outputs an approximation of its squareroot.

## Weekly Task 7 - week07 folder/es.py and test.txt
